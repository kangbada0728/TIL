# 15강. 코틀린에서 배열과 컬렉션을 다루는 방법

## 1. 배열

- 별로 쓰지 않지만 문법을 알고있자.

```java
int[] array = {100, 200};

for (int i = 0; i < array.length; i++) {
	System.out.println("%s %s", i, array[i]);
}
```

```kotlin
fun main() {  
    val array = arrayOf(100, 200)  
  
    for (i in array.indices) {  
        println("${i} ${array[i]}")  
    }
    for ((idx, value) in array.withIndex()) {  
        println("$idx $value")  
    }
}
```
